# Installation

## Requirements

- Go 1.25 or later. Tork Web's own `go.mod` requires it, and the baseline
  matters in one place you will notice: `omitzero` on a struct tag, which is
  what makes [`Optional[T]`](/docs/2.fundamentals/8.optional-values) disappear
  from a response when it holds nothing.
- Nothing else. The runtime imports only the standard library.

## Add the module

```sh
go get github.com/tork-go/web
```

The import path is `github.com/tork-go/web` and the package is `tork`:

```go
import "github.com/tork-go/web"

app := tork.New()
```

The two names differ because the repository is one of several under the same
organisation — `github.com/tork-go/orm` is the sibling ORM — while the package
you write against is the framework itself. Most editors fill the import in for
you; if yours does not, an explicit alias reads fine too:

```go
import tork "github.com/tork-go/web"
```

## Verify

A file that compiles and serves is the whole check:

```go
package main

import (
	"context"
	"log"

	"github.com/tork-go/web"
)

type Health struct {
	Status string `json:"status"`
}

func health(context.Context) (Health, error) {
	return Health{Status: "ok"}, nil
}

func main() {
	app := tork.New()
	app.GET("/health", health)
	log.Fatal(app.Run(":8080"))
}
```

```sh
go run .
curl -s localhost:8080/health
# {"status":"ok"}
```
