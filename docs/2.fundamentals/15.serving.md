# Serving

## Run

```go
if err := app.Run(":8080"); err != nil {
	log.Fatal(err)
}
```

`Run` serves until the process is asked to stop. SIGINT and SIGTERM begin a
graceful shutdown: the listener closes, requests already in flight get up to ten
seconds to finish, the singleton cleanups your providers registered run in
reverse, and `Run` returns nil. Anything else — a port already in use, a mistake
in your declarations — comes back as an error. An application served some other
way runs those cleanups through
[`App.Close`](/docs/2.fundamentals/14.dependencies#cleanup).

## Serve

`Serve` is `Run` with the stop signal supplied by the caller, which is what to
use when something else in the process owns the lifecycle:

```go
ctx, stop := signal.NotifyContext(context.Background(), os.Interrupt)
defer stop()

if err := app.Serve(ctx, ":8080"); err != nil {
	log.Fatal(err)
}
```

## ServeListener

When something other than an address decides where the server listens — a socket
handed over by the supervisor, a TLS listener, or a test that wants the port
assigned to it rather than chosen in advance:

```go
listener, err := net.Listen("tcp", "127.0.0.1:0")
if err != nil {
	log.Fatal(err)
}
log.Printf("listening on %s", listener.Addr())

if err := app.ServeListener(ctx, listener); err != nil {
	log.Fatal(err)
}
```

The listener is closed by the time it returns, including when the application
turns out not to build.

## Handler

`Handler` returns the `http.Handler` without listening:

```go
handler, err := app.Handler()
if err != nil {
	log.Fatal(err)
}
```

This is where every declaration is finally checked — options against the site
they were written at, paths against their prefixes, handler signatures against
what the framework knows how to call, and the whole route table against itself
for duplicates. All of it is reported together, so one build fixes one round of
mistakes:

```
items/router.go:24: GET /items: parameter 1: field ItemID reads the path
    parameter "item_id", but GET /items has no {item_id} in it
items/router.go:31: tork.Summary is not valid on a router
```

Because it is an ordinary `http.Handler`, an application can also be mounted
inside something else:

```go
mux := http.NewServeMux()
mux.Handle("/api/", handler)
mux.Handle("/", staticFiles)
```

## Testing

`Handler` is what a test wants: no port, no goroutine, no waiting for a server
to come up.

```go
func TestReadItem(t *testing.T) {
	app := tork.New()
	app.Include(items.Router())

	handler, err := app.Handler()
	if err != nil {
		t.Fatalf("build: %v", err)
	}

	rec := httptest.NewRecorder()
	handler.ServeHTTP(rec, httptest.NewRequest("GET", "/items/1", nil))

	if rec.Code != http.StatusOK {
		t.Fatalf("status %d: %s", rec.Code, rec.Body)
	}
}
```

Two options make responses reproducible enough to compare whole:

```go
app := tork.New(
	tork.Clock(func() time.Time { return fixed }),
	tork.Logger(slog.New(slog.NewTextHandler(io.Discard, nil))),
)
```

`Clock` holds the timestamp in every error body still; `Logger` keeps the
framework's own logging out of the test's output.

Asserting that a bad declaration *is* refused works the same way — build, and
read the error:

```go
func TestPathFieldMustMatchTheRoute(t *testing.T) {
	app := tork.New()
	app.GET("/items", readItem)   // readItem reads {item_id}

	_, err := app.Handler()
	if err == nil || !strings.Contains(err.Error(), "has no {item_id} in it") {
		t.Errorf("build error = %v", err)
	}
}
```

## Routes

`Routes` resolves the tree and returns the finished routes, in declaration
order, without building a server:

```go
routes, err := app.Routes()
for _, route := range routes {
	fmt.Println(route.Method, route.Path, route.OperationID, route.Tags)
}
```

It is what the OpenAPI builder will read, and what a test asserts against when
it wants the route table rather than a server.

## Configuration

Everything that changes how requests are served is an option on `tork.New`:

| Option | Effect |
|---|---|
| `Title`, `Description` | describe the API |
| `Logger(*slog.Logger)` | where unexpected failures are reported |
| `Clock(func() time.Time)` | the source of error timestamps |
| `MaxBodyBytes(int64)` | the largest request body, 1 MiB by default |
| `StrictBodies()` | reject unknown JSON body fields |
| `OnError(mapper)` | translate an error into a response |
| `WriteErrorsWith(writer)` | replace the error wire format |
| `Use(middleware...)` | middleware for every route |
| `Tags(...)` | tags for every operation |
